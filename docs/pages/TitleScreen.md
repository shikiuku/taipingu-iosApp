# TitleScreenドキュメント

## 概要
アプリの起動直後に表示されるメインタイトル画面。
ユーザーをゲームの世界観へと迎え入れ、開始ボタンを提供します。

## 主なコンポーネント
- **ロゴエリア**: アプリ名「打（タイピング）」とサブタイトルを表示。
- **スタートボタン**: 押下することで `CourseSelect` 画面へ遷移。
- **フッター**: iPad Landscape 最適化済みであることを示すテキスト。

## ロジック
- `useRouter`: Expo Router を使用して `CourseSelect` へのナビゲーションを管理。

## スタイル
- 背景: `AppColors.wood.dark` (#5D4037) を使用。
- テキスト: `AppColors.text.gold` や `white` を使用し、和風かつ高級感のある配色。
- レイアウト: 中央配置の構成。
