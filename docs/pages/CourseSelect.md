# CourseSelectドキュメント

## 概要
ゲームを開始する前に、難易度（コース）を選択する画面。
各コースには料金と制限時間が設定されています。

## 主なコンポーネント
- **コースカード**: 3,000円、5,000円、10,000円の各コースを選択するためのボタン。
- **コース情報**: 各コースの制限時間（60秒、90秒、120秒）を表示。

## 音声フィードバック
- **決定音**: `cursor.mp3` (コース選択時)。
- **戻る音**: `cancel.mp3` (タイトルへ戻る時)。

## ロジック
- `useGameStore / startCourse`: コース情報をストアにセットし、ステータスを `playing` に変更。
- `useRouter`: 選択後に `GameScreen` へ遷移。

## スタイル
- 背景: `AppColors.wood.dark` を使用。
- カード: ホバー（押下）感のあるインタラクティブなボタン配置。
- レイアウト: 水平方向にコースを並べた iPad 向けレイアウト。
