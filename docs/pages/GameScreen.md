# GameScreenドキュメント

## 概要
寿司屋のカウンターを舞台とした、和風でプレミアム感のあるタイピング画面。
静止したディスプレイボートに表示される単語を時間内に次々と打っていく「禅」のスタイルを採用。

## 構造 (コンポーネント構成)
`src/screens/GameScreen/index.tsx` は以下のサブコンポーネントに分割されています。

- `GameHeader`: タイマー、スコア、コース名、中断ボタンを表示。
- `SushiDisplay`: 寿司ネタ（漢字、かな）、ローマ字入力ガイドを表示（アニメーションなしの即時レスポンス設計）。
- `GameScreen`: 全体の状態管理、音声再生、レイアウトをコントロール。

## 主な機能
- **単語管理**: 各コースに基づいたワードリストからランダムに出題。
- **入力判定**: `useTypingLogic` フックを使用し、低遅延で入力を照合。
- **音声フィードバック**:
    - 単語完了時: `word_complete.mp3`（音ズレ防止のため非同期で先行再生）。
    - 中断時: `cancel.mp3`。
- **タイマー管理**: 制限時間終了で結果画面へ自動遷移。
- **レイアウト安定化**: ステータスバーを非表示にし、キーボード表示時もレイアウトが崩れないよう固定。

## スタイル管理
- すべての色は `src/constants/theme.ts` の `AppColors` を経由して管理。
- 背景: `wood.deep` などの高級感のある木目調を基調。
- レイアウト: iPad の横画面（Landscape）に完全最適化。
