# Taipingu - Sushida Style Typing Game for iPad

iPadに最適化された、和風（寿司屋カウンター）テーマのタイピングゲーム。

## 🛠 技術スタック
- **Framework**: Expo (Managed Workflow)
- **Language**: TypeScript
- **State**: Zustand
- **Animation**: React Native Reanimated

## 📜 開発の掟 (Crucial Rules)
1. **画面別カプセル化**: 画面固有のものは `src/screens/[Name]/` に閉じ込め、共通部品のみ `src/components/` に置く。
2. **ドキュメント駆動**: `PROJECT_DOCUMENT.md` を唯一の正解とし、AIが迷わないように常に更新する。
3. **iPad最適化**: iPadの広い画面を活かしたレイアウトを行う。
